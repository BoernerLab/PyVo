Ampel¬∂

Aufgabenstellung¬∂

Ein Kraftfahrzeug n√§hert sich einer Verkehrsampel mit verminderter Geschwindigkeit. Beim Umschalten der Ampel auf Gr√ºn wird das KFZ w√§hrend der Zeit \(t_{1}\) gleichm√§√üig mit \(a\) beschleunigt und legt dabei die Wegstrecke \(s_{1}\) zur√ºck.
Wie gro√ü sind die Geschwindigkeiten \(v_{0}\) und \(v_{1}\) am Anfang und am Ende der Beschleunigungsphase?

Gegeben¬∂

  • Beschleunigung \(a = 0,94~\frac{m}{s¬≤}\)

  • Zeit \(t_{1} = 5,3~s\)

  • Wegstrecke \(s_{1} = 60~m\)

Gesucht¬∂

Geschwindigkeit in Metern pro Sekunde [\(\frac{m}{s}\)]

  1. \(v_{0}\)

  2. \(v_{1}\)

Skizze¬∂

Physikalischer L√∂sungsweg¬∂

F√ºr die Berechnung von \(v_{0}\) und \(v_{1}\) ben√∂tigen wir das Weg-Zeit-Gesetz f√ºr die gleichf√∂rmig beschleunigte Bewegung:

\( s(t) = \frac{a}{2} \cdot{} t^2 + v_0 \cdot{} t + s_0 \;\;| s_0 = 0\)

\( s_1(t) = \frac{a}{2} \cdot{} t_{1}^2 + v_0 \cdot{} t_1 \)

Dadurch ergeben sich f√ºr \(v_{0}\) und \(v_{1}\) folgende Formeln:

\( v_{0} = \frac{s_{1}}{t_{1}} -\frac{a}{2} \cdot{}t_{1}\)

\( v(t) = a \cdot{} t + v_{0}\)

\( v_1(t) = a \cdot{} t + \frac{s_{1}}{t_{1}} -\frac{a}{2} \cdot{}t_{1}\)

Berechnung \(v_0\):

\(v_{0}(t_{1}) = \frac{60~m}{5,3~s} - \frac{1}{2}\cdot 0,94 \frac{m}{s^2} \cdot 5,3~s = 8,83~\frac{m}{s} ‚âà 32~\frac{km}{h}\)

Berechnung \(v\) mithilfe der zuvor bestimmten Geschwindigkeit \(v_0\):

\(v_{1}(t_{1}) = 0,94 \frac{m}{s^2} \cdot 5,3~s+ 8,83~\frac{m}{s^2} = 13,81~\frac{m}{s} ‚âà 50~\frac{km}{h}\)

Berechnung \(v\) ohne vorher bestimmte Geschwindigkeit \(v_0\):

\(v_{1}(t_{1}) = 0,94 \frac{m}{s^2} \cdot 5,3~s + \frac{60~m}{5,3~s} - \frac{1}{2} \cdot 0,94 \frac{m}{s^2} \cdot 5,3~s = 13,81~\frac{m}{s} ‚âà 50~\frac{km}{h}\)

Implemetierung in Python¬∂

Zuerst m√ºssen einige Bibliotheken importiert werden, die f√ºr Berechnungen, Visualisierung, etc. notwendig sind.

import matplotlib.pyplot as plt
import pandas as pd
import numpy as np

Nun k√∂nnen f√ºr die oben gezeigten Formeln durch PyVo-Funktionen definiert werden, mit denen im Anschluss die Ergebnisse f√ºr z.B. \(v_0\) und \(v_1\) berechnet werden k√∂nnen.

# Funktion zur Berechnung von v0
def berechne_v0(a, t, s):
    return (s/t) - (0.5*a*t)

# Funktion zur Berechnung von v1
def berechne_v1(a, t, s):
    return (a * t) + (s/t) - (0.5*a*t)

# allgemeine Funktion zur Berechnung der Geschwindigkeit v
def berechne_v(a,t, v0):
    return (a * t) + v0

# allgemeine Funktion zur Berechnung des Weges s
def berechne_s(v0, t, a):
    return (v0 * t) + ((a/2) * t**2)

Nachdem die Formeln definiert wurden, k√∂nnen nun die gegebenen Werte als Variablen definiert werden, diese der Funktion √ºbergeben und die Ergebnisse direkt ausgeben werden.

# Variablen definieren
a = 0.94 # m/s¬≤
t = 5.3  # s
s = 60   # m

# Funktionsaufruf zur Berechnung von v0
v0 = berechne_v0(a, t, s)
# Ausgabe des Ergebnisses unter Beachtung der signifikanten Stellen der Eingabeparameter (hier 2)
print(f"v0 = {round(v0,2)} m/s")

# Funktionsaufruf zur Berechnung von v1 ohne v0 zu kennen.
v1 = berechne_v1(a, t, s)
print(f"v1 = {round(v1,2)} m/s (Rechenweg 1, ohne vorher bestimmte Geschwindigkeit v0)")

# Funktionsaufruf zur Berechnung von v1 mithilfe des zuvor bestimmten v0.
v1 = berechne_v(a, t, v0)
print(f"v1 = {round(v1,2)} m/s (Rechenweg 2, mithilfe der zuvor bestimmten Geschwindigkeit v0)")

Graphische Darstellung¬∂

Nun k√∂nnen die Zusammenh√§nge der gleichm√§√üig beschleunigten Bewegung f√ºr die gegebenen Parameter visualisiert werden. Hierf√ºr werden f√ºr definierte Zeitschritt die Beschleunigung \(a\), die Geschwindigkeit \(v\) und zur√ºckgelegte Wegstrecke \(s\) abh√§ngig von \(t\) bestimmt.
Der folgende Code berechnet Beschleunigung, Geschwindigkeit und Weg f√ºr die gegebenen Werte:

Beschleunigung \(a_0 = 0,94~\frac{m}{s¬≤}\)
ben√∂tigte Zeit \(t = 5,3~s\)
zur√ºckgelegter Weg \(s_1 = 60~m\) nach zur√ºckgelegter Zeit \(t\)

# Definition der Startparameter
a0 = 0.94 # m/s^2
weg = 60 # m
t_gesamt = 5.3 #s

# Hier definieren wir einen Beobachtungszeitraum, der die √ºber die 5.3s hinaus geht und definieren dabei automatisch t0 und t1.
zeitschritte = np.arange(0,t_gesamt*2,0.1).round(1)
t1 = round(((t_gesamt*2) - (t_gesamt/2)),1)
t0 = round(t_gesamt/2, 1)

# Nun definieren wir eine Tabelle, in der wir die gemessenen Werte erfassen.
messwerte = []

# Initialisieren der Variablen und Berechnung von v0.
v0 = berechne_v0(a0, t_gesamt, weg)
s = 0
a = 0
for t in zeitschritte:
    # da nur zwischen t0 und t1 beschleunigt wird, ist die Beschleunigung 
    # nur zwischen t0 und t1 gleich der gegebenen Beschleunigung und ansonsten 0.
    if (t >= t0) and (t <= t1):
        a = a0
        v0 = berechne_v(a,0.1,v0) # 0.1 ist Œît
        s = s + berechne_s(v0,0.1,a) # s in Œîs
        messwerte.append([t-t_gesamt/2, a, v0, s]) # Hier f√ºgen wir die aktuelle Beobachtung unseren Messwerten hinzu
    else:
        a = 0 # au√üerhalb von t0 und t1 wird nicht beschleunigt.
        s = s + berechne_s(v0,0.1,a)
        messwerte.append([t-t_gesamt/2, a, v0, s])
        
#Als letztes werden die Messwerte in ein Pandas DataFrame konvertiert. Das macht die Visualisierung einfacher.
tabelle = pd.DataFrame(messwerte, columns=['time', 'a', 'v', 's'])

Graphische Darstellung unter Variation der Anfangsbedingung¬∂

Der Vorteil einer Programmiersprache wie Python ist es, dass die gegebenen Werte ver√§ndert oder auch die Geschwindigkeiten f√ºr mehrere Startparameter berechnet werden k√∂nnen. In folgendem Beispiel werden verschiedene Beschleunigungen \(a_{0,i}\) gegeben und dabei die Geschwindigkeiten \(v\) sowie die zur√ºckgelegten Wegstrecken \(s\) bestimmt.

Als erstes erfolgt die Definition verschiedener Beschleunigungen.

# Definition verschiedener Beschleunigungen
a0_vektor = [0.94, 1.94, 2.94, 3.94]

# Anhand der Zeit ergeben sich t0 und t1
zeitschritte = np.arange(0,t_gesamt*2,0.1).round(1)
t1 = round((t_gesamt*2) - (t_gesamt/2),1)
t0 = round(t_gesamt/2, 1)

Jetzt k√∂nnen in einer Schleife f√ºr verschiedene Beschleunigungen \(a\) alle Geschwindigkeiten \(v\) und Wegstrecken \(s\) in Abh√§ngigkeit von der Zeit \(t\) bestimmt werden:

# Definition einer Messwerttabelle
messwerte2 = []

# Jetzt wird durch die Iteration des a0_vektors f√ºr alle a_0 die Geschwindigkeit, Weg und v0 berechnet.
for a0_i in a0_vektor:
    v0 = berechne_v0(a0_i, t_gesamt, weg)
    s = 0
    a = 0
    for t in zeitschritte:
        if (t >= t0) and (t <= t1):
            a = a0_i
            v0 = berechne_v(a,0.1,v0)
            s = s + berechne_s(v0,0.1,a)
            messwerte2.append([t-t_gesamt/2, a, v0, s, a0_i])
        else:
            a = 0
            s = s + berechne_s(v0,0.1,a)
            messwerte2.append([t-t_gesamt/2, a, v0, s, a0_i])
